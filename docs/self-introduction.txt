自律型AIエージェント基盤の研究・開発
使用技術：LangGraph、GitHub Actions、Claude Code、OpenClaw、Discord、Telegram、各種LLM API

概要：
受注からヒアリング、開発、納品、請求、着金までの業務プロセスを自律的に支援・実行するAIエージェント基盤の研究・開発を実施。
単一機能の自動化ではなく、企業ごとに異なる課題に応じてエージェント構成を切り替えながら、継続的に課題解決を実行する「実行主体」としてのシステムを構想。
現在は調査・実験段階として、DiscordやTelegramを入口としたオーケストレーション構成や、LangGraph・OpenClawを用いた指示制御、GitHub Actions + Claude Codeによる開発エージェント連携の検証を進めている。

担当業務：
自律型AIエージェントのユースケース設計・要件整理
Discord / Telegram を起点としたマルチチャネル指示基盤の検討・実装
LangGraph を用いたエージェントのオーケストレーション設計
OpenClaw を活用したタスク指示・役割分担アーキテクチャの検証
GitHub Actions + Claude Code を用いた開発エージェント連携フローの構築
自律実行に必要な状態管理、タスク分配、実行監視の設計検討
継続的に課題へ適応するエージェント構成の研究・プロトタイプ開発

バックエンド開発を中心に、要件整理、API設計、DB設計、実装、テストまで一貫して担当してきました。Laravel、Python、TypeScript を用いた開発経験があり、業務要件を踏まえたシステム設計や、保守性・拡張性を意識した実装を得意としています。特に、データベース設計、業務ロジック実装、外部API連携、認証認可を含む業務システム開発に強みがあります。

インフラ面では、AWSを活用したシステム構築経験があり、API Gateway、Lambda、S3、DynamoDB、Cognito などを用いたサーバーレスアーキテクチャの実務に加え、近年は EKS を用いた Kubernetes 環境の構築や、EKS、SQS、Aurora Serverless、Dapr を組み合わせたクラウドネイティブなデータ管理アプリケーションの設計・実装にも取り組んでいます。アプリケーション開発に留まらず、アーキテクチャ全体を踏まえた設計・構築に対応できる点を強みとしています。

また、AIを活用したアプリケーション開発では、技術調査から設計、実装まで一貫して担当した経験があり、新しい技術を実際の業務要件へ落とし込むことが可能です。加えて、生成AIやAIエージェント領域の技術検証にも継続的に取り組んでおり、自律型AIエージェント基盤の研究・プロトタイプ開発を進めています。

フロントエンドについても、React、Next.js、Vue.jsを用いた実装経験があり、バックエンドを主軸としつつ必要に応じて幅広い領域に対応可能です。今後は、バックエンドおよびクラウド領域を軸に設計力をさらに高め、より複雑で高付加価値なシステム開発に貢献していきたいと考えています。
